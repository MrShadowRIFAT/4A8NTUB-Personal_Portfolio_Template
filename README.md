## 🎯 About This Project

A production-grade personal portfolio website showcasing professional work, skills, and achievements. Built with vanilla HTML, CSS, and JavaScript—no frameworks, no dependencies. Fast, lightweight, and optimized for performance.

**This portfolio is for viewing and evaluation only. Unauthorized download, modification, or redistribution is prohibited.**

---

## ✨ Features

- ✅ **Responsive Design** - Mobile-first approach, works on all devices
- ✅ **Modern UI/UX** - Clean, professional interface with smooth interactions
- ✅ **Dark Mode Support** - Built-in dark/light theme toggle
- ✅ **Performance Optimized** - Fast loading, minimal dependencies
- ✅ **SEO Ready** - Proper meta tags, structured data, semantic HTML
- ✅ **Smooth Animations** - Elegant transitions and entrance effects
- ✅ **Interactive Elements** - Custom cursor, parallax, wave effects
- ✅ **Contact Integration** - Professional contact section with form

## 🎨 Customization

The portfolio uses CSS custom properties and a color system for easy theming:

- **Colors**: Edit `css/colors.css` to change the color scheme
- **Typography**: Fonts configured in main stylesheet
- **Dark Mode**: Toggle via `css/darkMode.css`
- **Content**: Update HTML files with your own content
